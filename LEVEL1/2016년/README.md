## 2016년

### 문제설명

2016년 1월 1일은 금요일입니다. 2016년 a월 b일은 무슨 요일일까요?  
두 수 a ,b를 입력받아 2016년 a월 b일이 무슨 요일인지 리턴하는 함수, solution을 완성하세요.  
요일의 이름은 일요일부터 토요일까지 각각

> SUN,MON,TUE,WED,THU,FRI,SAT

입니다. 예를 들어 a=5, b=24라면 5월 24일은 화요일이므로 문자열 "TUE"를 반환하세요.

### 제한조건

- 2016년은 윤년입니다.
- 2016년 a월 b일은 실제로 있는 날입니다. (13월 26일이나 2월 45일같은 날짜는 주어지지 않습니다)

### 입출력 예

| a   | b   | result |
| --- | --- | ------ |
| 5   | 24  | "TUE"  |

## 🔥 내 풀이

해당 문제는 처음을 어떻게 짜서 문제를 시작해야할지 고민한 문제다.. 무려 레벨1임에도 이런 거에는 머리가 안 굴러가는 것 같았다.  
결국에는 검색의 힘을 빌려 어떻게 푸는 문제인지 살펴보았다!

우선, 2016년은 윤년이라는 조건을 통해 2월이 29일까지만 있는 년도임을 알 수 있었다.  
각 날짜 문자열이 들어간 배열을 하나 만들어준다. 이때 인덱스 1 자리에는 금요일이 들어가있는데,  
문제에서 1월 1일이 금요일이라고 했으므로 좀 더 편하게 문제를 풀기 위해 인덱스 1자리에 FRI를 넣었다.

그 다음 2016년의 각 달이 몇 일까지 있는지 담아놓는 배열이 필요하다. (7, 8월은 모두 31일이므로 실수하지 않도록 주의!)

입력받은 달 a를 months 배열에 넣는다. 여기서! a-1을 하는 이유는 인덱스 규칙에 의해 입력한 a가 5월일 경우 인덱스 4로 찾아야 하기 때문에 -1을 해준다.  
배열의 처음부터 해당 달까지 일 수를 sum 함수를 통해 더해준다.  
그리고 추가로 입력받은 일 수 b를 더해준 값에 또 더한다.

일주일은 7일 간격이므로 7로 나눠준 나머지값을 days 배열을 찾을 때 넣어주면 해당 a월 b일이 언제인지 알 수 있다!!

```python
def solution(a, b):
    days = ['THU', 'FRI', 'SAT', 'SUN', 'MON', 'TUE', 'WED']
    months = [31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

    return days[(sum(months[:a-1])+b)%7]
```
