## 수박수박수박수박?

### 문제설명

길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요.  
예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

### 제한조건

- n은 길이 10,000이하인 자연수입니다.

### 입출력 예

| n   | return     |
| --- | ---------- |
| 3   | "수박수"   |
| 4   | "수박수박" |

## 🔥 내 풀이

규칙을 파악했을 때, 길이 n이 짝수일 때와 홀수일 때로 나누어 풀 수 있을 것 같았다.  
짝수의 경우엔 들어온 글자 길이 n을 2로 나누어 "수박"과 곱해주면 해당 길이 n만큼 수박이 반복되게 된다.  
홀수의 경우엔 짝수와 비슷하지만 마지막에 "수"로 끝나도록 문자 하나를 덧붙여야 한다!

```python
def solution(n):
  text = '수박'

  if n % 2 == 0:
    answer = text * (n // 2)
  else:
    answer = text * (n // 2) + '수'

  return answer
```
